Política de Privacidade

Fiz Plantão

Última atualização: 9 de novembro de 2025

1. Informações que Coletamos

O Fiz Plantão coleta e processa as seguintes informações:

  • Informações de Conta: Email, nome e foto de perfil (quando usa Google Sign-In)
  • Dados de Plantões: Locais de trabalho, datas, horários, valores e status de pagamento
  • Acesso ao Google Calendar: Para sincronizar eventos de plantão

2. Como Usamos Seus Dados

Utilizamos suas informações exclusivamente para:

  • Autenticar e identificar você no aplicativo
  • Armazenar e sincronizar seus dados de plantões entre dispositivos
  • Criar e atualizar eventos no seu Google Calendar
  • Melhorar a experiência do usuário

3. Uso do Google Calendar API

O Fiz Plantão solicita acesso ao Google Calendar para:

  • Criar eventos automaticamente quando você cadastra um plantão
  • Atualizar eventos existentes quando você edita um plantão
  • Remover eventos quando você deleta um plantão
  • Criar eventos de pagamento agrupados por data

Garantias:

  • Apenas eventos criados pelo app são modificados
  • Não acessamos, lemos ou modificamos outros eventos do seu calendário
  • Você pode revogar o acesso a qualquer momento em: myaccount.google.com/permissions

4. Armazenamento de Dados

  • Local: Dados armazenados no seu dispositivo usando Hive (banco de dados local)
  • Nuvem: Sincronizados com Supabase (PostgreSQL) para backup e acesso multi-dispositivo
  • Segurança: Conexões criptografadas (HTTPS/TLS)
  • Isolamento: Seus dados são privados - outros usuários não têm acesso

5. Compartilhamento de Dados

NÃO compartilhamos seus dados pessoais ou de plantões com terceiros, exceto:

  • Google (apenas para autenticação OAuth e sincronização de calendário)
  • Supabase (apenas para armazenamento e sincronização de dados)

Nunca vendemos, alugamos ou comercializamos suas informações.

6. Seus Direitos

Você tem direito a:

  • Acessar: Ver todos os seus dados armazenados
  • Corrigir: Editar informações incorretas
  • Deletar: Remover sua conta e todos os dados associados
  • Revogar: Cancelar acesso ao Google Calendar a qualquer momento
  • Exportar: Solicitar cópia dos seus dados

7. Retenção de Dados

  • Dados são mantidos enquanto sua conta estiver ativa
  • Ao deletar a conta, todos os dados são permanentemente removidos em até 30 dias
  • Backups podem ser mantidos por até 90 dias

8. Segurança

Implementamos medidas de segurança para proteger seus dados:

  • Autenticação Firebase com verificação de email
  • Conexões criptografadas (TLS/SSL)
  • Row Level Security no banco de dados
  • Tokens OAuth seguros para acesso ao Google Calendar

9. Menores de Idade

O app não é destinado a menores de 18 anos. Não coletamos intencionalmente dados de menores.

10. Alterações nesta Política

Podemos atualizar esta política ocasionalmente. Notificaremos sobre mudanças significativas através do app ou por email.
